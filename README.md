# 🛍️ Rogue Vogue

Welcome to **Rogue Vogue** — a modern, stylish online fashion store offering trendy clothes and shoes at affordable prices.

---

## 🌟 About Rogue Vogue

Rogue Vogue is a one-page e-commerce website designed to showcase and sell fashion items such as clothes and shoes. The platform focuses on simplicity, speed, and a smooth shopping experience, especially optimized for mobile users.

Customers can easily browse products, view prices, and place orders instantly via **WhatsApp** or a **Google Form**.

---

## 🚀 Features

* 📱 Fully responsive one-page design (mobile + desktop)
* 🖼️ Product gallery with images and pricing
* 💬 WhatsApp ordering with pre-filled product messages
* 📝 Google Form integration for structured orders
* ⚡ Smooth scrolling user experience
* 🎨 Clean and modern fashion-focused UI
* 📞 Contact and social media links in footer

---

## ⚙️ Setup & Usage

1. Download or copy the project files
2. Open `index.html` in your browser
3. Customize the following:

### 🔧 Update Phone Number

Replace:

```
254712345678
```

With your WhatsApp number in international format.

### 🔧 Update Google Form Link

Replace:

```
YOUR_FORM_LINK
```

With your actual Google Form URL.

### 🔧 Add Products

Edit the product array in the JavaScript section:

```javascript
{
  name: "Product Name",
  price: "Ksh XXXX",
  image: "image-url"
}
```
